{# ===== 6. Setup guide (collapsible) ===== #}
TradeStation OAuth setup guide (one-time)
  1. Go to developer.tradestation.com
  2. Create an app — note Client ID and Client Secret
  3. Set redirect URI to http://localhost:5000/oauth/callback
  4. To get your initial refresh token, visit (replace YOUR_CLIENT_ID):
    https://signin.tradestation.com/oauth/authorize?response_type=code&client_id=YOUR_CLIENT_ID&redirect_uri=http://localhost:5000/oauth/callback&audience=https://api.tradestation.com&scope=openid%20profile%20MarketData%20ReadAccount%20Trade
  5. After login, copy the code from the redirect URL
  6. POST to https://signin.tradestation.com/oauth/token with grant_type=authorization_code, code=…, client_id=…, client_secret=…, redirect_uri=…
  7. Save the refresh_token from the response into .env as TS_REFRESH_TOKEN
  8. The app auto-rotates the refresh token on each connect.
