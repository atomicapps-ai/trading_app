# 8LQsR7B2vQ0 — "Best scalping strategy (no indicators, pure price action + market structure)"

Source: <https://www.youtube.com/watch?v=8LQsR7B2vQ0>

## Content — discretionary market-structure scalping (5-minute FX)
- Read the trend on the 1-hour (higher highs/lows vs lower highs/lows, or consolidation).
- Drop to the 5-minute, look for a **break of structure** (price breaks a previous higher-low
  / lower-high / pullback).
- After the break, **wait for price to come back and retest that broken level**, then enter
  in the break direction ("expect price to retest this level then shoot back down").
- Entry timing, stop placement, and targets are all shown by hand on the chart, no numeric rule.

## Verdict: REJECT — not mechanizable
Everything hinges on discretionary calls: which swing counts as "structure," when a retest is
"good enough," where the stop and target go. There is no fixed entry trigger, no defined stop,
and no defined target — the author draws each level by eye ("focus on what price is doing right
now"). This is the same class already rejected repeatedly in this run (discretionary
liquidity/structure scalps with no defined exits). No backtestable kernel beyond generic
break-of-structure retest, which isn't specified precisely enough to code faithfully.
Status: rejected
