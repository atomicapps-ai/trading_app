# bITIVwysCzM — "First 2 hours" opening-range strategy

Source: <https://www.youtube.com/watch?v=bITIVwysCzM>

## Content — opening-range (ORB) with multiple entry models
- Instead of the whole (large) range, mark the **high/low of the first candle of a new market
  session** (1 / 5 / 15-min) = the "opening range" — treated as high-liquidity external levels.
- Trade it within the first ~2 hours via one of several **entry models**: breakout, break-and-
  retest, or reversal — plus "confluences" to make it higher-probability. Entries, stops, targets
  and examples follow.

## Verdict: REJECT — opening-range ground already covered; entry is multi-model / discretionary
This is opening-range-breakout, an area this project already explored directly: the removed
opening-candle work (First-Hour-Conviction FHC-S1 failed; the `double_lock` opening pattern was
removed for not generalizing), and a 2-minute opening-range video was earlier rejected because
its daily analog is the live `coil_breakout`. The video doesn't commit to a single mechanical
rule — it offers breakout **or** retest **or** reversal plus discretionary "confluences" — so
there's no one faithful spec to test, and the mechanical core (opening-range break) is already
represented/known. No new separable edge to validate.
Status: rejected
